# Vimのmodelineを使ってファイルの折りたたみ設定を変更する

- modelineを利用すると、ページの上部もしくは下部に設定を記載して、そのファイルにだけ設定を反映させることができる
- 例えば、下記のように記載するとfoldの設定をmarkerに変更できる

```
" vim: foldmethod=marker foldlevel=1
```
