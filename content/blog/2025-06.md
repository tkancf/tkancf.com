# 2025-06-24

- tmuxで現在のディレクター名をwindow名にしたい

```
# ウィンドウ名の自動リネームを有効にする
set-window-option -g automatic-rename on

# ウィンドウ名のフォーマットを設定（ディレクトリ名のみ表示）
set-window-option -g automatic-rename-format '#{b:pane_current_path}'
```
